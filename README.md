# 🚀 Ambassify - Campus Ambassador Growth Dashboard

## 📌 Overview
Ambassify is a smart and interactive dashboard designed to track and analyze campus ambassador performance. It helps organizations monitor engagement, signups, and growth trends using real-time data visualization.

---

## 🎯 Features
- 📊 Dashboard with real-time data simulation
- 🏆 Leaderboard ranking system
- 📈 Growth trend analysis (Line Chart)
- 📊 Signup tracking (Bar Chart)
- 🥧 Contribution percentage (Pie Chart)
- 🔴🟢🟡 Performance indicator buttons
- 🤖 AI-powered chatbot for insights
- 📂 CSV/JSON file upload support
- 🔐 Login system (basic authentication)

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
- Chart.js

---

## 📊 Dashboard Functionalities
- Track ambassador signups and engagement
- Identify top performers
- Analyze growth trends
- Compare performance metrics
- Generate insights using chatbot
